Library Manager — Mini Project

Flask backend Tkinter GUI JSON persistence

This is a simple local library manager. The desktop GUI allows the user to view a list of media items (books, films, magazines), filter by category, search by exact name, add new items, and delete items. The backend is a Flask server that stores data persistently in data/library.json. The app runs locally on your machine and is intended for demonstration.

How to run (short)

  1. Activate the Python virtual environment: .\venv\Scripts\activate (Windows).
  2. Start backend: python backend/app.py.
  3. Open a second terminal and run GUI: python frontend/gui.py.
  4. Open this page: double-click website/index.html.
