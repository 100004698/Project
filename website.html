What the app does — simple explanation

Library Manager is a small desktop application to manage a personal collection of media items (books, films, magazines). It has two parts:

  • Frontend (desktop GUI): `frontend/gui.py` — built with Tkinter. Shows a list of items, a details pane, category filter, exact-name search, and buttons to create or delete items.
  • Backend (server): `backend/app.py` — built with Flask. Provides REST endpoints (list, search by name, get by id, create, delete) and stores data in `data/library.json`.

How it behaves (user flows)

  • View list: on start the GUI loads all items from the backend (example items: Ice and fire, Iron Man, Superman, Inception).
  • Filter: choose Book / Film / Magazine and press Filter — GUI asks backend for the filtered list.
  • Exact-name search: type the full exact name (e.g. Inception) and press Search — backend returns matching items.
  • Create: click Create, fill the modal (name, date YYYY-MM-DD, author, category) — backend validates and saves the new item with a UUID id.
  • Delete: select an item and press Delete (confirmation shown) — backend removes the item from the JSON file.
Storage format: items are saved in `data/library.json` as a mapping from UUID → record. Example record fields: id, name, publication_date, author, category.

How to run (copy & paste — Windows)

Open two terminals. First start the backend, then the GUI. These commands assume you run them from the project root (the folder that contains `backend/`, `frontend/`, `data/`).

1) Install dependencies (one time)

pip install --upgrade pip
pip install flask requests pytest

2) Start the backend (terminal 1)

python backend\app.py
# Expected output: * Running on http://127.0.0.1:5000

3) Start the GUI (terminal 2)

python frontend\gui.py
# The Tkinter window will open and load items from the backend

If the GUI shows "Cannot connect to backend server", make sure the backend terminal shows it is running on http://127.0.0.1:5000. The GUI uses that exact URL (see `BASE = "http://127.0.0.1:5000"` at top of `frontend/gui.py`).
